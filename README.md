# بوت تنبيهات أسعار الكريبتو (Binance)

بوت تيليغرام يرسل لك تنبيهًا عند وصول عملة معينة إلى سعر محدد. **لا ينفّذ أي عملية شراء أو بيع** — للمراقبة فقط.

## 1. الحصول على توكن البوت
1. افتح تيليغرام وابحث عن `@BotFather`
2. أرسل `/newbot` واتبع التعليمات
3. احفظ الـ Token الذي يعطيك إياه

## 2. التثبيت محليًا (للتجربة)
```bash
cd crypto_alert_bot
pip install -r requirements.txt

# على Linux/Mac
export BOT_TOKEN="التوكن_هنا"
python bot.py

# على Windows (PowerShell)
$env:BOT_TOKEN="التوكن_هنا"
python bot.py
```

## 3. الأوامر داخل تيليغرام
| الأمر | المثال | الوصف |
|---|---|---|
| `/price` | `/price BTCUSDT` | عرض السعر الحالي |
| `/alert` | `/alert BTCUSDT 65000` | تعيين تنبيه عند وصول السعر |
| `/alerts` | `/alerts` | عرض التنبيهات النشطة |
| `/remove` | `/remove 3` | حذف التنبيه رقم 3 |

- رموز الأزواج تُكتب بصيغة Binance، مثل: `BTCUSDT`, `ETHUSDT`, `SOLUSDT`
- التنبيه يُحذف تلقائيًا بعد تفعيله مرة واحدة

## 4. الاستضافة الدائمة (24/7)
البوت يحتاج للعمل باستمرار حتى يصلك التنبيه في أي وقت. خيارات مجانية/رخيصة للبداية:
- **Railway** (railway.app) — الأسهل، يدعم Python مباشرة
- **Render** (render.com) — خطة مجانية لكن قد "تنام" بعد عدم النشاط
- **PythonAnywhere** — مناسب للمشاريع الصغيرة

في كل الحالات، تحتاج لتعيين متغير البيئة `BOT_TOKEN` من إعدادات المنصة (وليس كتابته في الكود مباشرة، حفاظًا على أمانه).

## 5. ملاحظات مهمة
- هذا البوت يستخدم بيانات Binance العامة فقط (لا يحتاج مفتاح API من حسابك)
- لأن السعر يُفحص كل 30 ثانية، قد يتأخر التنبيه قليلًا عن اللحظة الفعلية لتغير السعر
- التنبيهات تُحفظ في ملف `alerts.json` — إذا انتقلت لاستضافة جديدة، انقل هذا الملف معك للحفاظ على تنبيهاتك
